

# ECMAScript + JavaScript

历史

    网景开发了JavaScript，一年后微软又模仿JavaScript开发了JScript，
    为了让JavaScript成为全球标准，几个公司联合ECMA组织定制了JavaScript语言的标准，
    被称为ECMAScript(ES)标准。

区别

    ECMAScript是一种语言标准，
    JavaScript是网景公司对ECMAScript标准的一种实现。
