

# 用;表示语句结束

规范

    JavaScript语句以;结束，
    但是，JavaScript并不强制要求在每个语句的结尾加;

原理

    浏览器引擎会自动在每个语句的结尾补上;

注意

    让JavaScript引擎自动加分号在某些情况下会改变程序的语义，导致运行结果与期望不一致。
    

# 用{}表示语句块

规范

    语句块用{...}包裹

    注意花括号{...}内的语句具有缩进，通常是4个空格。
    缩进不是JavaScript语法要求必须的，但缩进有助于我们理解代码的层次

# 注释

单行注释

    以//开头直到行末的字符被视为行注释

块注释

    用/*...*/把多行字符包裹起来，把一大“块”视为一个注释
